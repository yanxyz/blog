# VSCode Terminal

`Ctrl+\`` 打开 terminal

macOS 和 Linux 使用 SHELL 环境变量。

Windows 使用 COMSPEC 环境变量，它通常指向 cmd.exe。在第一次打开 terminal 时，会提供一个 shells 列表选择。`terminal.integrated.shell.windows` 设置 shell，默认为 powershell。
