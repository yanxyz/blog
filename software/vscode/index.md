# VSCode

VSCode(Visual Studio Code) 是微软出品的一个轻量 IDE，与 Atom 一样基于 Electron，相比 Atom:

- 启动速度快一点
- 常用功能是现成的，比如：Emmet, formatting, Markdown preview，git，debugging 等

提示：本笔记只考虑 VSCode 最新版本，如果有不一致，先更新 VSCode。
