# Shell 语法

不同的 Shell 有不同的语法，这里主要关注 [POSIX Shell 语法](https://ynotes.github.io/shell/)。

- Bash，POSIX Shell 语法可以视作它的语法子集
- Dash，是 Ubuntu 默认的 Shell, 它兼容于 Posix。

## Basics

Shell 标识符（路径，变量，命令，关键字等）区分大小写。

```sh
# 注释
```
