# Sublime Text

Sublime Text 是一款轻量级编辑器。
它并不是 IDE，虽然可以用插件添加 git, debug 等功能，但是配置麻烦而且功能有限。
我只是将它作为快速编辑工具（这时插件越少越好），开发用 [VSCode](../vscode/index.md)。

Sublime Text 是付费软件，不过不注册没有功能限制，只是随机弹出对话框提醒注册。
