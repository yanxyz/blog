# PHP 变量

变量不用声明。

变量名字以 `$` 开始。

```php
$message = 'hello php';
echo $message;
```

变量的类型可以动态转换。


`isset()` 用于判断变量是否初始化。

- [PHP isset() vs empty() vs is_null()](https://www.virendrachandak.com/techtalk/php-isset-vs-empty-vs-is_null/)

## Constants

常量不以 `$` 开始。常量名字按惯例全大写。
