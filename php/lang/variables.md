# PHP 变量

PHP 没有声明变量语句。

PHP 变量以 `$` 开始，后面为变量名字。

`isset()` 用于判断变量是否初始化。

- [PHP isset() vs empty() vs is_null()](https://www.virendrachandak.com/techtalk/php-isset-vs-empty-vs-is_null/)

变量的类型是动态变化的。

函数内的变量为本地变量，PHP 一点不同是，全局变量不能直接在函数内使用。

## Constants

常量不以 `$` 开始。常量名字按惯例全大写。
