# Markdown

Markdown 是一种轻量级的标记语言。“轻量”是相对于 HTML、XML等标记语言而言。它可以说是程序文档的标准撰写工具。对于非程序员群体，比如网编，也是不错的工具。

## 对于普通用户

了解 [Markdown 语法](http://markdown.tw/)。通常网站编辑器会提供一份语法速查。

编辑器：

- Windows: [MarkdownPad](http://markdownpad.com/)
- Mac: [Mou](http://25.io/mou/)
