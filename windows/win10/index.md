## 任务视图

任务视图（Task View）即 虚拟桌面 virtual desktops

点击任务栏上的任务视图图标（WIN+TAB），打开任务视图，这里可以创建新的桌面，将程序拖动到其它桌面内

- WIN + CTRL + D: 创建新的桌面
- WIN + CTRL + LEFT/RIGHT: 切换上个或下个桌面
- WIN + CTRL + F4: 关闭当前的桌面，程序合并到前一个桌面
