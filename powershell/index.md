# PowerShell

PowerShell 是微软出品的 shell，建立在 .Net framework 上，当前版本可以跨平台（Windows, Linux,  macOS）运行。不同于一般的 shell（如 bash），PowerShell 的 pipeline 传递的是 .NET objects 而不是 strings。
