# Gem

Gem 是 Ruby 的包管理器。

[使用国内的 RubyGems 镜像。](https://gems.ruby-china.org/)

Gem 配置文件为 `~/.gemrc`（用户级别），为 YAML 格式。

[Gem 命令](http://guides.rubygems.org/command-reference/)

```bash
gem --help
gem help commands
gem list
gem install json
gem uninstall json
gem clean # 清除所有的旧版本
```
